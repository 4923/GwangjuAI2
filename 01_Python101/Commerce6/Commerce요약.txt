1. 프로그램 시연! 및 구현된 클래스 소개 
   ⓛ 로그인
    - 로그인 DB 불러오기, DB Upload
    - 사용자 정보확인
    - 회원가입 
	ID 중복 검사, 비밀번호 유효성 추가-> 회원가입 진행(입력한 비밀번호 동일 유무 파악+닉네임 중복 검사 후 가입)		
    - ID/PW 일치 확인 후 로그인 성공/실패 여부 확인
    - 로그인 한 회원 기준으로 회원탈퇴 진행
    - 회원 로그인 후 쇼핑 및 커뮤니티 게시판 이동(비회원일 시 회원가입 유도, 회원일 경우 회원탈퇴란 생성)
  
   ② 상품 구매
    - 장바구니 
	장바구니 속 현재까지 선택한 상품 품목 개수, 총 구매 금액 표현
	상품 전체 구입 or 부분 구입 여부 선택 (구입 여부 선택 후 '랜덤 보안코드' 생성)
    - 상품선택
	상품 분류 : 평점, 리뷰, 가격, 년도, 장르로 구분( 장르 선택시 장르에 맞춰 다시 상품분류 진행)
		   구매할 서적 기준
			<구매>선택 시    : '영수증 발급'으로 이동
			<비구매>선택 시 : 다시 '상품 분류'로 이동
			<장바구니>선택 시 : 선택 상품 장바구니 이동
	안전결제를 위한 랜덤 보안코드 생성 (일치하게 입력한 경우에만 구매 진행)

	영수증 발급( 주문 목록, 금액, 주소, 닉네임 포함)
             배송안내 : 요청사항 추가 여부 후 고객닉네임, 서적이름, 주소 포함한 배송 송장정보 표시
		
	장바구니 목록 캡처 후 이메일 발송
